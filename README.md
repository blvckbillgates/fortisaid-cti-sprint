# FortisAid Cyber Threat Intelligence Sprint

## Overview
This repository contains the deliverables from a Cyber Threat Intelligence (CTI)
sprint as part of the **CyBlack programme**. Our team produced an operational
threat landscape assessment for **FortisAid**, a fictional UK-based healthcare
technology organisation.

Using **OpenCTI** enriched with **AlienVault OTX** feeds, we analysed
healthcare-sector threats, national (UK) risks, recent victim profiles, and
politically motivated threat actors. The output is a full intelligence report
and an executive briefing deck designed to support security operations and
leadership decision-making.

## Key Findings at a Glance
| Area | Top Threats / Actors | Risk to FortisAid |
| :--- | :--- | :--- |
| Healthcare Sector | Akira, Interlock ransomware | Double extortion, service disruption, PHI exposure |
| UK National Landscape | Kraken ransomware, Candiru spyware | Operational downtime, covert surveillance |
| Recent Victims | Conti (gov), MintLoader (finance), PowerShell RAT (tech) | Valuable victimology insights for defence |
| Political Actor | APT29 (Cozy Bear) | Long-term espionage, supply-chain compromise |
